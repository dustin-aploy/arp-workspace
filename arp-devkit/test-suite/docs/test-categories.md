# Test Categories

## Schema tests
Validate that required ARP sections remain present and protocol-owned structure is enforced.

## Behavior tests
Exercise blocked, approval-gated, escalated, and budget-exhausted behavior through the reference runtime flow.

## Audit tests
Confirm that decisions produce audit artifacts with required fields, event types, and outcomes.

## Reporting tests
Confirm that runtime-generated daily reports remain protocol-shaped and that compliance reports contain the fields required by this suite.
