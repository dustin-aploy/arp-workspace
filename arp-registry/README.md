# ARP Registry

`arp-registry` models the publication and listing layer for the Agent Responsibility Protocol.

It references:
- the normative protocol from `../arp-spec`
- compliance evidence generated by `../arp-devkit/test-suite`
- reference execution/report concepts exercised by `../arp-devkit/runtime`

The registry does **not** redefine protocol semantics. It points to protocol versions and protocol-shaped evidence.
