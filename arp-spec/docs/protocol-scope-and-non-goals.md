# Protocol Scope and Non-Goals

## In scope

- normative responsibility definitions for organizational agents;
- machine-readable schemas;
- versioned protocol artifacts;
- examples and guidance for protocol adopters; and
- governance and RFC process for evolving the standard.

## Out of scope

- agent execution engines;
- benchmark runners;
- integration adapters;
- hosted registries;
- automatic certification systems; and
- vendor-specific policy tooling.
