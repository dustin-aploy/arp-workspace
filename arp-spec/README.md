# ARP Spec

`arp-spec` is the normative protocol target for the Agent Responsibility Protocol (ARP).

It is the source of truth for:
- ARP schemas
- protocol versions
- normative examples
- RFCs and protocol docs
- governance and contribution rules
- the packaged `arp_protocol` Python artifact

## Layout

```text
arp-spec/
  README.md
  LICENSE
  VERSION
  CONTRIBUTING.md
  GOVERNANCE.md
  pyproject.toml
  spec/
  schemas/
  examples/
  rfcs/
  docs/
  src/arp_protocol/
```

## Install locally

```bash
pip install -e ./arp-spec
```
